# **App Name**: Hiii

## Core Features:

- Direct Chat Access: Open the app directly to the chat screen, bypassing any login or setup screens.
- Real-time Messaging: Send and receive messages instantly with real-time synchronization between both users.
- Cloud Storage: Store messages securely in the cloud (Firestore or Firebase Realtime Database) to ensure both users see the same chat history.
- Auto-scroll: Automatically scroll to the latest message in the chat.
- Love Streak Counter: Track the number of days since the first chat and display it with a heart animation. This will be displayed via the use of the database service
- Custom Emoji Reactions: Allow users to react to messages using custom emoji.
- Voice Notes: Users can send voice notes and see a heart pulse animation while recording.

## Style Guidelines:

- Background Gradient: Soft pink to peach gradient (#f9e3e3 to #f7c7b6) to create a romantic ambiance.
- Message Bubbles (You): Chocolate tone (#d98e73) for the user's messages.
- Message Bubbles (Partner): Creamy frosting white (#fff6f8) for the partner's messages.
- Accent / Buttons: Dark cocoa (#a05d5d) for buttons and interactive elements.
- Highlights / Heart Icon: Lily-pink (#e75480) for highlights and the heart icon.
- Text: Coffee brown (#4a2e2e) for all text elements.
- Body and headline font: 'PT Sans' for a warm and modern feel.
- Use a heart send button ❤️ and custom romantic emoji reactions.
- Implement cute, rounded message bubbles for a friendly and intimate feel.
- Display a heart animation when a message is sent and a heart pulse animation while recording voice notes.
